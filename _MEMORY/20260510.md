## v1.12 Design Decisions

Goal: align the aggressive mechanical pullback definition before trying ML again.

Changes versus v1.11:

- Align indicator params to live Super Structure defaults:
  - SuperTrend factor 4.0, ATR period 12
  - ADX length 12
  - CCI length 12, source `hl2`
- Keep aggressive as DEMA pullback scalper, RR 1:1.
- Use DEMA100 as directional filter, with DEMA50/100/200 distance/slope features retained.
- Remove macro features from baseline (`oil_return`, `us10y_change`, `dxy_return`) to avoid regime-overfit before baseline is known.
- Change pullback proximity from price-percentage (`st * 1.002` / `st * 0.998`) to ATR-based band:
  - `pullback_band = max(0.5 pts, atr * 0.25)`
- Candidate rules:
  - Long: ST direction stays bullish (`-1`), close > DEMA100, close > ST, low touches `ST + band`, green candle.
  - Short: ST direction stays bearish (`+1`), close < DEMA100, close < ST, high touches `ST - band`, red candle.
- SL: ST +/- 1.0 point buffer.
- TP: 1R.
- Max hold: 100 bars.
- Timeout now marked-to-market at final close, not automatic loss.
- Added audit fields:
  - `entry_price`, `sl_price`, `tp_price`, `exit_ts`, `exit_price`, `exit_reason`, `hold_bars`, `risk_pts`, `pnl_pts`, `pnl_usd`
  - DEMA distances/slopes, `entry_adx`, `entry_cci`, `cci_abs`, `rsi_7`, wick/body/range features, `touch_distance_atr`, `hour_utc`, `dow`, `session_cluster`.
